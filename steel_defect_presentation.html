Dataset Split NEU-DET
  • Pre-divided train and validation folders in the dataset
  • Programmatically split validation 50/50 into val and test — no files moved on disk
  • Stratified split ensures equal class representation in each half
  • Perfectly balanced: 300 images per class, no class imbalance
Preprocessing Steps PIPELINE
  • Grayscale → RGB conversion (ResNet50 requires 3-channel input)
  • Resize to 224×224 pixels (ResNet50's native input size)
  • Normalization: pixel values 0–255 → 0.0–1.0
  • CLAHE applied to enhance fine crack and texture visibility
CLAHE (Contrast Limited Adaptive Histogram Equalization): Divides the image into small tiles and equalises each independently, boosting local contrast. This makes subtle surface textures — especially fine cracks and pitting — much more visible to the network.
Augmentation Strategy
🔄
Rotation ±20°
Defects appear at any orientation on the production line. Random rotation makes the model orientation-invariant.
↔️
Horizontal Flip
Doubles effective training data by mirroring images, helping the model generalise to unseen orientations.
🔦
Brightness Shift
Simulates factory lighting variation. Prevents the model from relying on absolute brightness as a feature.
Key Insight: Aligning preprocessing exactly with ResNet50's expectations — correct normalization and RGB channel order — was what took us from ~65% to 98% accuracy. Preprocessing was the single biggest lever.
We prepared the images carefully before feeding them to the model. The key challenge was that the images were grayscale but ResNet50 expects 3-colour RGB input. We converted them and also applied CLAHE to enhance texture contrast. Aligning our preprocessing exactly with ResNet50's expectations was what took us from 65% to 98% accuracy.
P3 · Model & Architecture
Technical Heart &
Architecture Explained
Transfer Learning with ResNet50 — why we chose it, how it works, and why it outperformed the alternatives.
Transfer Learning Analogy: Think of it like hiring an experienced photographer who has studied a million images, and just teaching them what steel defects look like. ResNet50 already knows shapes, textures, and edges — we only teach the final classification step.
What is ResNet50 BACKBONE
  • 50-layer deep convolutional neural network
  • Pre-trained on ImageNet: 1M+ images, 1000 classes
  • Skip connections solve the vanishing gradient problem in deep networks
Custom Classification Head OUR LAYERS
  • GlobalAveragePooling2D — compresses 7×7 feature maps to a single vector
  • Dense(256, relu) — learns high-level defect feature combinations
  • Dropout(0.5) — randomly disables neurons to prevent overfitting
  • Dense(6, softmax) — outputs 6 class probabilities that sum to 1

P4 · Training & Evaluation
Training Story &
Results + Debugging
Two-stage training, smart callbacks, and the debugging journey that took accuracy from ~65% to 98.3%.
Stage 1 — Warm-Up FROZEN
  • ResNet50 backbone frozen — weights untouched
  • Only the custom head layers are trained
  • Higher learning rate: 1e-3
  • Goal: train the new head without corrupting ImageNet features
  • ~10–15 epochs until head converges

  # first training phase
  base_model.trainable = False

  inputs = layers.Input(shape=(IMAGE_SIZE[0], IMAGE_SIZE[1], 3))
  x = base_model(inputs, training=False)
  x = layers.GlobalAveragePooling2D()(x)
  x = layers.BatchNormalization()(x)
  x = layers.Dense(256, activation="relu")(x)
  x = layers.Dropout(0.5)(x)
  outputs = layers.Dense(NUM_CLASSES, activation="softmax")(x)

  return model, base_model
Stage 2 — Fine-Tuning UNFROZEN
  • Top ResNet layers unfrozen — full model trains together
  • Very low learning rate: 1e-5
  • Adapts deep features to steel defect textures
  • Must use low LR to avoid destroying pre-trained weights
  • Where accuracy jumps to 98.3%

def unfreeze_top_layers(base_model: Model) -> None:
  base_model.trainable = True

  for layer in base_model.layers[:-UNFREEZE_TOP_LAYERS]:
    layer.trainable = False

  for layer in base_model.layers:
    if isinstance(layer, layers.BatchNormalization):
      layer.trainable = False

def prepare_fine_tuning(model: Model, base_model: Model) -> Model:
  unfreeze_top_layers(base_model)
  return compile_model(model, FINE_TUNE_LEARNING_RATE)
Training Callbacks
🛑
EarlyStopping
Monitors val_loss. Stops training automatically if no improvement for N epochs. Prevents overfitting and wasted compute.
📉
ReduceLROnPlateau
Detects when training stalls and automatically reduces the learning rate by a factor. Helps the model escape flat regions.
💾
ModelCheckpoint
Saves the model weights only when val_accuracy improves. Guarantees the final saved model is the best-ever checkpoint.
Debugging Journey — From 65% to 98%
1
Problem: Grayscale images fed as-is
ResNet50 expects 3-channel RGB. Grayscale images were duplicated across channels but normalization was misaligned. Accuracy stuck at ~65%.
2
Fix: Correct preprocessing pipeline
Matched exact normalization to ResNet50's expected input range. Added CLAHE preprocessing. Accuracy jumped to ~90%+.
3
Fix: Two-stage fine-tuning
Added Stage 2 with very low LR (1e-5) to fine-tune top ResNet layers on steel textures. Accuracy reached 98.3%.
Final Results: 98.3% test accuracy · F1 score > 0.98 across all 6 classes · Near-perfect diagonal in confusion matrix · Adam optimizer with categorical cross-entropy loss
Training happened in two stages. First we trained only our custom layers with ResNet frozen. Then we unfroze the top layers and fine-tuned everything with a very low learning rate. Our initial accuracy was around 65% — after fixing the preprocessing pipeline and adding two-stage fine-tuning, it rose to 98.3%. The callbacks automatically stopped training when improvement plateaued and saved only the best weights.

Grad-CAM Explainability
What is Grad-CAM? EXPLAINABILITY
  • Uses mathematical gradients to find which parts of the image most influenced the decision
  • Gradients computed at the last convolutional layer of ResNet50
  • Renders as a red-yellow-green heatmap overlay on the original image
  • Heatmap concentrates on the actual defect, not the background
